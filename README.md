# Dotfile

Here is my dotfile for setting up my mac.

## Note
### Fonts:

   - SFMono Nerd Font: https://github.com/epk/SF-Mono-Nerd-Font
   - Liga SFMono Nerd Font: https://github.com/shaunsingh/SFMono-Nerd-Font-Ligaturized
    
### Starship:
  Change default config location to `~/config/starship` folder

  ```bash
  echo export STARSHIP_CONFIG=~/.config/starship/config.toml >> ~/.config/zsh/exports.zsh
  ```
